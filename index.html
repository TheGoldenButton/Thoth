Starting your local LLM

LM Studio
1. Download at lmstudio.ai and load a model
2. Go to Developer tab → enable Local Server
3. Make sure CORS is on in server settings
Runs at localhost:1234 — no terminal needed

Ollama
1. Install at ollama.com
2. Pull a model:
ollama pull llama3.2
3. Start serving:
OLLAMA_ORIGINS=* ollama serve
Runs at localhost:11434

OpenRouter requires an API key from openrouter.ai — works without running anything locally.
